# Infrastructure

Infrastructure is intentionally documentation-only for this milestone.

The platform is expected to use AWS CDK in TypeScript in a future task, with planned resources including API Gateway, Lambda, DynamoDB, Cognito, S3, SQS, SES, CloudFront, Route 53, and ACM.

No CDK package, deployable stack, AWS resource definition, AWS account connection, or deployment command exists in this repository yet.
